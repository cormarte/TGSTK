\doxysubsection{Detailed Description}
Linear-\/quadratic model for computing cell survival to radiation therapies. 

\mbox{\hyperlink{classtgstk_linear_quadratic_tumour_cell_survival_image_filter}{tgstk\+Linear\+Quadratic\+Tumour\+Cell\+Survival\+Image\+Filter}} computes a survival normalised tumour cell density field $c_s(\bar{r})$ from user-\/provided initial normalised tumour cell density field $c_0(\bar{r})$ and dose map $d(\bar{r})$ using the linear-\/quadratic model \cite{rockne_2010} \+:

\[ c_s(\bar{r}) = \mathrm{e}^{-\left(\alpha \, d(\bar{r})+\beta \, d^2(\bar{r})\right)} \, c_0(\bar{r}) \]

where $\alpha$ and $\beta$ are respectively the linear and quadratic radio-\/sensitivity coefficients.

The initial normalised tumour cell density field $c_0(\bar{r})$ and the dose map $d(\bar{r})$ can be specified by the user as \href{https://vtk.org/doc/nightly/html/classvtkImageData.html}{\texttt{ vtk\+Image\+Data}} objects. The linear and quadratic radio-\/sensitivity coefficients $\alpha$ and $\beta$ can also be specified.

\begin{DoxyWarning}{Warning}
As the C\+U\+DA implementation performs in single precision floating point, a loss of precision may occur. 
\end{DoxyWarning}
