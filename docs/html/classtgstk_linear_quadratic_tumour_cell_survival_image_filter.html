Detailed Description

Linear-quadratic model for computing cell survival to radiation therapies.

tgstkLinearQuadraticTumourCellSurvivalImageFilter computes a survival normalised tumour cell density field \(c_s(\bar{r})\) from user-provided initial normalised tumour cell density field \(c_0(\bar{r})\) and dose map \(d(\bar{r})\) using the linear-quadratic model [6] :

\[ c_s(\bar{r}) = \mathrm{e}^{-\left(\alpha \, d(\bar{r})+\beta \, d^2(\bar{r})\right)} \, c_0(\bar{r}) \]

where \(\alpha\) and \(\beta\) are respectively the linear and quadratic radio-sensitivity coefficients.

The initial normalised tumour cell density field \(c_0(\bar{r})\) and the dose map \(d(\bar{r})\) can be specified by the user as vtkImageData objects. The linear and quadratic radio-sensitivity coefficients \(\alpha\) and \(\beta\) can also be specified.

Warning
As the CUDA implementation performs in single precision floating point, a loss of precision may occur.
